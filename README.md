A modern, highly interactive, and visually stunning personal portfolio designed to showcase my journey as a **Frontend Developer** and **Computer Science Student**. The UI/UX is built with a dreamy "Lilac & Plum" color palette, incorporating glassmorphism, fluid animations, and responsive layouts.

🔗 **Live Demo:** [https://personal-web-portfolio-one.vercel.app/](https://personal-web-portfolio-one.vercel.app/)

---

## ✨ Key Features

- 🎨 **Unique "Lilac & Plum" Aesthetic**: A cohesive and premium color system combining soft purple backgrounds with vibrant amber/fuchsia gradients.
- 🚀 **High Performance**: Built with Vite + React for lightning-fast Hot Module Replacement (HMR) and optimized production builds.
- 💫 **Interactive Animations**: Utilizes `framer-motion` and Tailwind CSS keyframes for floating elements, gradient text, smooth scroll reveals, and bouncy hover states.
- 📱 **Fully Responsive**: A mobile-first design approach ensuring the portfolio looks perfect on all device screen sizes.
- 🧩 **Modular Architecture**: Component-driven development with isolated, reusable React components (`HeroSection`, `ProfileSection`, `ProjectsSection`, etc.).

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (version 18+ recommended) and `npm` installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NgocHuyen2309/Personal_Web_Portfolio.git
   cd Personal_Web_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open [http://localhost:5173](http://localhost:5173) in your web browser.

---

## 🔧 Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/):

1. Push your code to a GitHub repository.
2. Go to your Vercel Dashboard and click **Add New > Project**.
3. Import this repository.
4. Vercel will automatically detect the **Vite** framework. Leave the default settings and click **Deploy**.
